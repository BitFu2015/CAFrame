输出控制接口

函数

void drv_output (uint8_t port, uint8_t val)
 开关量输出控制函数，在驱动(driver.c)中必须实现此函数。

详细描述

输出控制接口函数drv_output以框架统一的方式控制单片机IO口的输出行为，
这些IO口被编号成IO_Y1,IO_Y2 ...
drv_output函数实现编号对应的IO的输出控制行为。
