详细描述

定时器操作十分简单，一个最简单的示例如下(app.c)：
void TimerEvent1(void)
{
LED_FLASH;//发光管闪烁
}
void InitApp(void)
{
frm_install_timer_event(0,300,TimerEvent1);//安装0号定时器事件处理函数,定时触发周期为300ms
// ... 其它初始化操作
}
注解
要注意frm_install_timer_event函数第一个参数要小于在Cfg.h中定义的TIMER_MAX_COUNT
