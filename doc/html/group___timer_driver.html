详细描述

定时器操作十分简单，一个最简单的示例如下(app.c)：

EVENT_TIMER g_Timer1;
void TimerEvent1(void)
{
LED_TOGGLE;//发光管控制端口取反，使其闪烁
}
void InitApp(void)
{
frm_create_timer(0,&g_Timer1);//框架中创建定时器
frm_install_timer_event(0,300,TimerEvent1);//配置定时器并安装事件处理函数,定时触发周期为300ms
// ... 其它初始化操作
}
